1. Contar os Caracteres em uma String

Para contar o número de caracteres em uma string, usamos a função len().

# Código em Python
frase = "Python é incrível"
numero_de_caracteres = len(frase)
print(numero_de_caracteres)  # Saída: 16

Resposta: A frase "Python é incrível" tem 16 caracteres, incluindo o espaço.

2. Converter String para Maiúsculas

Para converter todos os caracteres de uma string para maiúsculas, usamos o método upper().

# Código em Python
frase = "python é incrível"
frase_maiuscula = frase.upper()
print(frase_maiuscula)  # Saída: PYTHON É INCRÍVEL

Resposta: "PYTHON É INCRÍVEL"

3. Converter String para Minúsculas

Para converter todos os caracteres de uma string para minúsculas, usamos o método lower().

# Código em Python
frase = "PYTHON É INCRÍVEL"
frase_minuscula = frase.lower()
print(frase_minuscula)  # Saída: python é incrível

Resposta: "python é incrível"

4. Contar Ocorrências de um Caracter

Para contar quantas vezes um determinado caractere aparece em uma string, usamos o método count().

# Código em Python
frase = "Python é incrível"

print(contador)  # Saída: 2

Resposta: O caractere "i" aparece 2 vezes na frase.

5. Verificar se uma Palavra está na Frase

Para verificar se uma palavra existe dentro de uma string, podemos usar o operador in.

Resposta: "incrível" está presente na frase, então o resultado é True.

56. Calcular Aluguel de Carro

# Código em Python

total_km = km_percorridos * custo_km

print(f"Custo dos km rodados: €{total_km:.2f}")
print(f"Total a pagar: €{total_pagar:.2f}")
